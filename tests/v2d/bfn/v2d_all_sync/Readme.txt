Steps to create executables
-------------------------------------------------------------------------------------------------------------------------------------------------------
gmake clean; gmake

Steps to run the test
-------------------------------------------------------------------------------------------------------------------------------------------------------
This script will take 1 arguments and script should be run in below format

python3 v2d_all_sync.py <drf_clock>

drf_clock can be with in the range [0,1]

ex: python3 v2d_all_sync.py 0

Note: This information is for the FPGA model ABC_18A_0p6_R1.0_ww12
