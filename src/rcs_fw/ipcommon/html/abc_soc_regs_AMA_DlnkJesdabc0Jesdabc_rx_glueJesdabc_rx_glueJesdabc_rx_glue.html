+0x00000030 Register(32 bit) RX_CONV_SMPL_INTRLV_CTRL
RX Convereter Sample interleaving control
RX Convereter Sample interleaving control. To support 64 streams in single JESD link, samples from converter interface need to be interleaved before accumulating required samples and writing to RSB.
16 groups of 4 consecutive converters are formed ( group 0 - conv 0-3 ...). In each group, first two samples from each converter are selected. An 8x8 sample cross bar rearrages them into new set of 4 converters and 2 samples per conveter.
Each output sample can be selected from any of the 8 input samples as defined in this register. This 24-bit control is common across all 16 converter groups.
