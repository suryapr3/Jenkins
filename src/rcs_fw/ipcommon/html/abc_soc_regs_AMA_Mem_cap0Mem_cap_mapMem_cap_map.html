POWER_DOWN

The MEM_CAP_MEM memory block contains a MEM_CAP_MEM 1.75MByte memory which is divided into 7 sectors. These sectors can be powered down individually using these register bits. Here bit 0 corresponds to Sector 0 and bit 6 corresponds to Sector 6. When the bit is set, the sector corresponding to that bit is powered down and is not available for normal operation. These memories come up in a powered down state (i.e. the corresponding control bit is set to 1) after reset. The user must actively turn off the power down state in order to use this memory. This is done to ensure that the chip is configured to use the minimum amount of power.

ECC Config
Configuration register for the MEM_CAP AW_ecc_refresh module. Each of these configuration registers supports 1 AWMW_ABC78_MEM_CAP_sabram_rw_pd_4096x156 instance. Each of these instances supports 128 bits of user data, split into 32 bit slices. Each 32-bit slice is protected by its own 7 bit ECC word.
ECC configuration bits for data[31:0] are contained in bits 6:0 and are suffixed with _0.
ECC configuration bits for data[63:32] are contained in bits 14:8 and are suffixed with _1.
ECC configuration bits for data[95:64] are contained in bits 22:16 and are suffixed with _2.
ECC configuration bits for data[127:96] are contained in bits 30:24 and are suffixed with _3.
Bits 31,23,15, and 7 are reserved.
Each group of 4 addresses provides configuration support for a single sector. Address NNN0 configures ECC for bits 127:0, Address NNN4 configures ECC for bits 255:128, Address NNN8 configures ECC for bits 383:256, and Address NNNC configures ECC for bits 511:384. If Address X is the base address for the ECC Config registers, then X + Y*16 is the first of 4 addresses associated with memory sector Y.

Status register for the MEM_CAP AW_ecc_refresh module. This is connected to AW_ecc_refresh.sbe_o, which is SBE error level output and set on first error and held till cleared by *_ECC_ERR_CLR. Each instance of this register provides status for either one or two memory sectors.
