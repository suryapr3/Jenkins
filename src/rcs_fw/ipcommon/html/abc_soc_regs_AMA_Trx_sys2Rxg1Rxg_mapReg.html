+0x0000001c Register(32 bit) CSR_ERROR
RXG CSR Error Register
This register contains parity error indications for the RXG coefficient registers.
The bits in this register will clear themselves when the underlying register has been re-written with the correct parity (i.e when CSR_CONFIG.TEST_CTRL=0).
