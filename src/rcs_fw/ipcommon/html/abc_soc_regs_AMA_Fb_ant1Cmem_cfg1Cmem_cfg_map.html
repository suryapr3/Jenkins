+0x00000054[+=0x4] Register(32 bit) CMEM_ECC_ERR_INFO[8]
CMEM ECC Error Information Register
This is the information register for CMEM memory ECC error handling. For ECC the CMEM is broken up into 8 slices . Each is protected by ECC.
Please note that the CMEM is a 256 bit wide memory and the AXi interface is only 64 bits. Each 64 bit AXI access causes the entire 256 bits of CMEM to be read and ECC checked. This means that an ECC Error can occur on any one of the four 64 bit regions in CMEM. In short you can have ECC errors from anywhere in the 256 bit CMEM data even those that are not being read by the AXI. Note the ECC logic will indicate which 64 bits of the CMEM the error occurred.

Array dimensions:
 Count=8, Start=0, Step=1
