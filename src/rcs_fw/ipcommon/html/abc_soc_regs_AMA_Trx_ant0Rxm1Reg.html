PVAL Sample Window
Selects Size of the Sample Window based on the number of valid Clock cycle or Blocks.
The maximum sample size is 32,767 valid block.
The Minimum Sample size mustbe programmed based on the clock rate.
For 983.04 MHz the Minimum Sample Window Size is 32.
For1.475GHz the minimum sample Window is 48.
If the user mistakenly attempts to program thissregister with a value less than 32, it will be set to 32.
A read of this register in this case will read back a value of 32.

+0x00000018 Register(32 bit) RXM_PVAL_MEAS_OUT
Peak Value Measurment
Peak Value Valid, Average Power Valid, PTI1, Peak Value and Average Power.
1. Peak Value Valid, Average Power Valid and PTI1 flags are sticky.
2. Register is cleared on Read.
3- Note: See PTI_ALARM1_CNT register for the number of times the alarm was asserted.
4. The Formula for Calculating RXM Average power in terms of DSs is given below.
RXM_Peak _dBFS = 20*log10(RXM_Peak*2^-12)

In this system 0dBFS peak means saturation. The strongest CW tone average power would be -3dBFS.
