+0x00000040 Register(64 bit) bridge_rbc_rbm_63_30_rbm_error_mask_value
Error event mask register
Non-secure register, Programmable bits only for non-fatal errors as defined by P_FATAL_ERRORS. 1 b1: masks corresponding error from interrupt, 1 b0: allows corresponding error to cause interrupt. Fatal error bit positions are fixed to 1 b1 so that they do not exercise the non-fatal interrupt line. Fatal errors are non-maskable and directly exercise the fatal interrupt line.
