USE_SAME_SWG
Allows MOD1 to use the same SWG as MOD0
When set to 1, MOD1 and MOD0 both use SWG0. When this bit is set, to upconvert to 2 different bands, CONJ_1 = !CONJ_0. Otherwise, when set to 0, MOD 1 uses SWG1 and MOD0 uses SWG0.

Reset: hex:0x0;

RW

CONJ_1

DMOD_EN_PER_SAMPLE
per sample enable function,
Allows the user to indicate which incoming samples are enabled. The incoming data stream has a signle valid bit arriving with the up to 8 IQ samples every clockcycle. The user can choose to enable only the N lower samples. This allows the data path for the 8-N upper samples to be clock gated.
