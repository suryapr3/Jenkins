+0x00000010 Register(32 bit) TXM_PEAK_VALUE_MEAS_OUT
Peak Value Measurement Output
Peak Value Valid and Peak Value Measurement.
Valid bits are sticky. Measurment only update when measurments are valid.
All bits are cleared on read.
The formula for expressing Peak value in terms of DBs is given below.
TXM_Peak _dBFS = 20*log10(TXM_Peak*2^-11)
In this system 0dBFS peak means saturation. The strongest CW tone average power would be -3dBFS.

+0x0000001c Register(32 bit) TXM_GLOBAL_PEAK_VALUE_MEAS
Global_Peak Value
Represents the Global Peak Value is collected over multiple sample windows starting from reset or initialization state.
Data for the Global Peak Value is collected over a constant window size of eight valid clocks (i.e. blocks).
The global peak value is measured starting from one of three events listed below.
- Release from Reset
- Global Clear Release
- Down link (DLINK) signal transition high with the option to flush in pogress data i.e. initialzie the data path.
- Data Path Initialization signal is set low after being set high.
